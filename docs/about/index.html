About Ferminal

Ferminal or i like to call it F*ck Terminal is a custom wrapper cli.

Ferminal was created for time efficient and easy to use cli wrapper. You might be realize that writing command on terminal is kinda suck. For example you use: 

git commit -m "Commit Message"

That's really long, so on Ferminal, you can just type: 

gim Commit Message

It was same to the long one, but with efficient command. I know this was really hard to use at first, but if you use it for a long time, how much time efficient you get?

This project is not using subprocess, so it might be a bug sometime when you use it. If you want to contribute, you can check our github: Ferminal.
