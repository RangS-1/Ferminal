Installation

v1.3.4

Hello to installation section. Before we start, we must install prerequisites!

Prerequisites

Here are requirements for Ferminal:

  • python (v3.13 or higher for core program)
  • python-build (required for build package)
  • python-wheel (required for build package)
  • python-installer (required for install package)
  • python-colorama (required for external dependencies)
  • Git (required for external dependencies)

Installation Steps

You can install ferminal on different operating system, here's how:

Arch Linux

If you want to install on Arch Linux, you can use the following command

Note
Ferminal for Arch Linux only available on Arch User Repository. So you must install Yay or Paru. After that, you can use them by the following command:

Yay

yay -S ferminal

Paru

paru -S ferminal

Don't forget to check PKGBUILD file on Ferminal AUR Repository

Windows

If you want to install on Windows, here's how:

  1. Open this GitHub Release: Ferminal Release.
  2. Click on assets.
  3. Click file exe.
  4. Wait until download done.

After you install the exe file, then just double click on it and there you go, you can use it now.
