Git

v1.3.4

Welcome to Git Section. In here, you will learn how to use Ferminal to be more efficient using Git.

Prerequisites

sudo pacman -S --needed base base-devel git

Commands

Here are the Git commands supported by Ferminal, grouped by purpose:

Setup & Repository

gic <repository-url>   # git clone
gin                       # git init

Snapshot & Review

gia .                    # git add
gis                      # git status
gim <commit-message>      # git commit -m ""
gil                      # git log
gid <file-or-branch>      # git diff
girh                     # git reset HEAD
girs <file>               # git restore

Branch & Merge

gib <branch-name>        # git branch
gibd <branch-name>       # git branch -d
gio <branch-name>        # git checkout
giob <branch-name>       # git checkout -b
gie <branch-name>        # git merge

Remote & Sync

gip <remote>              # git pull
gih <remote>              # git push
gif <remote>              # git fetch
gir <command>             # git remote

Tag & History

gig <tag-name>            # git tag
gish <commit>             # git show
gibl <file>               # git blame

Stash & Cleanup

gist                     # git stash
gista                    # git stash apply
gifd                     # git clean -fd
