Commands

v1.3.4

Below is the full command reference available in Ferminal, grouped by category for easier reading.

Prerequisites

No special setup is required beyond running Ferminal from a terminal. Just enter the commands below as written.

Commands

Here are the commands supported by Ferminal, grouped by purpose:

File & Directory

d <path>                  # change directory
l                        # list directory contents
la                       # list directory contents including hidden files
k <folder>                # create a folder
m <folder>                # remove a folder
f <file>                  # create a file
r <file>                  # delete a file
v <path>                  # move a file or folder
c <file>                  # copy a file
cr <folder>               # copy a folder recursively
u <file>                  # show file contents

System & Utility

x                        # exit the app
b                        # clear the screen
e <text>                  # echo text
h                        # show help menu
w                        # show current working directory
z                        # pause for 30 seconds
p <host>                  # ping a host
